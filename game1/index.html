  • Вода + Ветер — туман/дождь, на сцене даёт шторм.

  • Земля + Растение — корни и лоза.
  • Свет + Барьер — световой купол.
  • Огонь + Вода — паровая завеса.
  • Земля + Огонь — магма.
  • Земля + Вода — грязь и рост.
  • Свет + Растение — бурное цветение.
  • Свет + Вода — призматический луч.
  • Вода + Лозы — цветущие лозы с цветами.
  • Земля + Вода + Свет — цветение для семечка.
